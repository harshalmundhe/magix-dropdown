== Description ==

Magix Dropdown is a simple 2 select dropdown plugin.

== Installation ==

1. Go to Plugins > Add New
2. Search for Magix Dropdown and Install it.
3. To add new dropdown go to , head over to **Magix Dropdown** > **Add New** menu in you WordPress's left vertical menu


== Use ==

Use the shortcode [md] for dispalying the dropdown on your page
